## 需求分析

- 文件选择:
  - Button 组件 点击触发;
- 文件处理：
  - 提取出选中文件(e.target.files);
  - 将文件给予 axios 处理（写一个文件处理函数，也是连接三大函数的核心 code）;
    - 函数只是处理交互，axios 逻辑还是无附加 code 的。
      - upload 逻辑：照猫画虎罢了;
      - 各类情况回调函数：画蛇添足罢了;
