## API

通过设置 Menu 、MenuItem 和 SubMenu 的属性来产生不同的按钮样式，推荐顺序为:

Menu: defaultIndex -> onSelect -> mode。

**Menu 的属性说明如下：**

<ArgsTable of={Menu} />

MenuItem: disabled。

**MenuItem 的属性说明如下：**

<ArgsTable of={MenuItem} />

SubMenu: title。

**SubMenu 的属性说明如下：**

<ArgsTable of={SubMenu} />
