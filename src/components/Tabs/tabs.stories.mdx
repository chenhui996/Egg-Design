# Tabs Component

选项卡切换组件。

## Tabs header

```js
import { Tabs } from "egg-design";
```

## API

通过设置 Tabs 和 TabItem 的属性来产生不同的标签页样式，推荐顺序为:

Menu: defaultIndex -> onSelect -> mode。
